DateSense

DateSense is a smart web application that helps users track the expiry dates of their products like food, medicines, and cosmetics. By entering a product name and its expiry date, users can save items and instantly see how many days are left until expiration. It improves safety, reduces waste, and simplifies daily management of perishable items.

Advantages:
• Avoids using expired products
• Reduces waste and saves money
• Keeps track of multiple items in one place
• Easy and accessible via web browser

Future Scope:
• SMS/email reminders before expiry
• Image upload for each product
• User login and personalized dashboard
• Mobile app version with barcode scanner
